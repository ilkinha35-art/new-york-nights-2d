# New York Nights 2D

Jogo inspirado em Nova York, com Times Square, Central Park e trânsito fluindo em 2D.

## Recursos
- Mapa de Manhattan em 2D
- Times Square com iluminação
- Central Park explorável
- Trânsito com carros em movimento

## Tecnologias
- Pygame (Python)
- Sprites desenhados à mão ou importados

## Como rodar
1. Clone o repositório
2. Instale pygame (`pip install pygame`)
3. Execute `python scripts/main.py`
